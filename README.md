# Blackjack Academy Pro

Kompletní offline PWA vytvořená od nuly.

## Funkce
- lokální AI Coach založený na pravidlech a analýze chyb
- trénink celého stolu
- rychlost rozdávání jako v kasinu
- denní a týdenní grafy
- XP, levely, kariéra, mise a achievementy
- Hi-Lo, KO, Omega II, Halves, Zen Count a Red Seven
- pravidla kasina: počet balíčků, S17/H17, 3:2/6:5, penetrace, double, surrender
- analýza každého rozhodnutí podle basic strategy
- 3D animace a realistické karty
- offline režim a instalace na plochu

## Nahrání na GitHub Pages
1. Nahraj všech 7 souborů do kořene repozitáře.
2. Settings → Pages.
3. Deploy from a branch.
4. main / root.
5. Save.

Poznámka: AI Coach je lokální chytrý pravidlový systém, ne cloudový jazykový model. Díky tomu funguje offline a nevyžaduje API klíč.
